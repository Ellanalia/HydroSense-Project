# HydroSense-Kenya
### ICS 2207 Scientific Computing – Capstone Project | February – May 2026

## Overview
A Python scientific computing system for smart irrigation, water balance simulation, and climate-aware decision support for a multi-zone demonstration farm in Kenya.

**Central Question:** Given weather and soil-sensor data, how can we model water availability, estimate water deficit, simulate future soil moisture, and recommend an efficient irrigation plan that minimises water use without exposing crops to moisture stress?

## Quick Start
```bash
pip install -r requirements.txt
jupyter notebook notebooks/Level_1_Problem_Framing.ipynb
```
Run notebooks in order (Level 1 → 6). Each notebook is self-contained and reads from `data/raw/`.

## Core Model
`S(t+1) = S(t) + R(t) + I(t) − ET(t) − D(t)`

ET = max(0, 0.12·T + 0.35·W + 2.4·Solar − 0.025·H)
